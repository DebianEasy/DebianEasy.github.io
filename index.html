DebianEasy

A Debian-based Linux distro aimed to be friendly out-of-box with openSUSE-like Btrfs support

Download Github Repository Discussion

DebianEasy (or DebEasy) is a Linux distro basing on Debian Stable, which is majorly forked from SpiralLinux and designed to provide a friendly alternative live installation option for Debian with better out-of-box hardware support and openSUSE-like Btrfs experience.

feature

  • Newer kernel and firmwares from Debian backports repository and kernel.org for new haredware supports
  • Out-of-box multi kernel supports
  • Btrfs as default file system and a better Btrfs subvolumn layout
  • Preinstalled grub-btrfs to provide an openSUSE-style grub interface
  • Debian backports and nonfree repository defaultly enabled
  • An additional custom repository for more kernel versions, backports and even some applications
  • Some settings to make it friendly to new users

    • Q&A

  • Why another Debian installer?
    • There are three main goals for DebianEasy: first, to make Debian intaller more friendly; second, to provide a good out-of-box btrfs supports like openSUSE; at last, to provide a better out-of-box hardware supports.

  • What is the relation and difference from SpiralLinux and DebianEasy?
    • DebianEasy is a fork of SpiralLinux. SpiralLinux does not contain its own repository. On the other hand, DebianEasy has an own addtional repository. Besides, DebianEasy remove some applications so that users can choose which applications to use and whether to use flatpak version.

  • How can I get spin of other desktop environment that not provieded here?
    • You can use source of iso which can be found in out github. Note that the source is not complete. You need to add addtional files following the Readme.
